# cryptDeck
University work (ргр1)

Как добавить новые шифры в программу (ворк ин прогресс):
1. В инклюды вручную прописываем относительный путь до файла;
2. В енумы добавляем пункты для своего шифра;
3. В функциях seqEncode и seqDecode в свитчах используем свои функции.
4. В билдерах добавляем свои библиотеки по образу и подобию других
